# Anti-Goals

What this system is NOT and explicitly rejects:
1. **No Material/Tailwind Defaults**: Avoid using generic drop-shadows, default roundness, or blue highlight focus rings.
2. **No Nature Stickers**: Do not place arbitrary illustrations of leaves, trees, or animals. Nature is rhythm and growth algorithm only.
3. **No Smooth Transitions**: Avoid generic linear or cubic ease-in transitions on element states. All state shifts are immediate.
4. **No Green Tick Status Language**: Do not represent success as a checkbox or green check icon. Status is organic change.
