# Atmosphere

Atmosphere defines the ambient environment of the system:
1. **Stepped breathing grid**: A subtle, slow-pulsing background grid of hairlines desynchronized using LCG values.
2. **Warm background bloom**: Dynamic radial gradients layer soft colors (cream, amber, red) behind the sheet.
3. **Deterministic Ambience**: Everything is mathematically seeded to avoid clock drift and keep build results byte-identical.
