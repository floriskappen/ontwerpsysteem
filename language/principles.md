# Principles

1. **Tokens and Value Layer as Truth**: Everything visual in the system originates from tokens or recipes.
2. **Vendor-Neutral & Portable**: Raw values are open and machine-readable (DTCG JSON), not bound to any proprietary vendor or tool.
3. **Intent over Appearance**: Tokens and component roles are named for meaning (e.g. `color.text.soft`), not literal styling or values (e.g. `color.gray-600`).
4. **Three Tiers, Never More**: primitive -> semantic -> component.
5. **Theme by Swap**: Swapping themes means changing semantic token mapping targets, never copying files.
6. **Tactile Integrity**: Layouts follow a print aesthetic—warm paper, ink, multiply blend modes, grain, and stepped clocks.
